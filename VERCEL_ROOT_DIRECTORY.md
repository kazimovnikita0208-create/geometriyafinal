# 🔍 Как найти Root Directory в Vercel

## 📍 Где искать настройку Root Directory

### Вариант 1: В разделе "General" (самый верх страницы)

1. Убедитесь, что вы в разделе **Settings** → **General**
2. **Прокрутите страницу ВВЕРХ** - настройка "Root Directory" обычно находится в самом верху страницы, **ПЕРЕД** секцией "Vercel Toolbar"
3. Ищите секцию с названием **"Root Directory"** или **"Project Root"**

### Вариант 2: В разделе "Build and Deployment Settings"

1. В левом сайдбаре нажмите на **"Build and Deployment"** (второй пункт после "General")
2. В этом разделе должна быть настройка **"Root Directory"**
3. Нажмите **Edit** рядом с ней
4. Введите: `frontend`
5. Нажмите **Save**

### Вариант 3: Если настройки нет вообще

Если настройка "Root Directory" отсутствует, это может означать, что:
- Проект был создан через Git интеграцию, и Vercel автоматически определил структуру
- Нужно использовать файл `vercel.json` (который уже создан)

В этом случае:
1. Убедитесь, что файл `vercel.json` находится в корне репозитория
2. Сделайте новый commit и push
3. Vercel автоматически применит настройки из `vercel.json`

---

## 🎯 Что делать прямо сейчас

**Попробуйте следующее:**

1. **Прокрутите страницу Settings → General в самый ВЕРХ** - настройка Root Directory должна быть там первой
2. Если не нашли - перейдите в **Settings → Build and Deployment** и ищите там
3. Если всё равно не нашли - сделайте скриншот всего раздела "General" (с прокруткой вверх и вниз)
