🌐 ДЛЯ ТЕСТИРОВАНИЯ В TELEGRAM:

1. Установите ngrok: https://ngrok.com/download

2. Запустите в новом терминале:
    ngrok http 3000

3. Скопируйте HTTPS URL (например: https://abc123.ngrok-free.app)

4. Откройте simple-bot.js и измените строку:
    const MINI_APP_URL = 'https://abc123.ngrok-free.app';

5. Перезапустите бота (Ctrl+C, затем npm start)

6. Откройте бота в Telegram → "Открыть приложение"!
