# ✅ Исправлено отображение занятий в абонементах

## 🐛 Проблема:
В карточках абонементов со статусом `pending` не показывалась информация о количестве занятий в формате "1/1 Занятий (осталось/всего)".

Показывалось только: **"Ожидает подтверждения"**

## 🔧 Решение:
Теперь информация о занятиях показывается **для всех статусов** абонементов:
- `pending` - Ожидает подтверждения
- `confirmed` - Активна
- `rejected` - Отклонена
- `expired` - Истекла
- `completed` - Завершена

## 🎯 Что показывается:

### Для ВСЕХ статусов:
- ✅ **"X/Y Занятий (осталось/всего)"** - всегда видно

### Для pending:
- ✅ Количество занятий: **"1/1 Занятий"**
- ✅ Вместо дней: **"- Ожидает активации"**
- ✅ Блок: **"Ожидает подтверждения"** (желтый)
- ❌ Нет прогресс-бара (еще не активен)

### Для confirmed:
- ✅ Количество занятий: **"8/12 Занятий"**
- ✅ Дни до окончания: **"25 Дней до окончания"**
- ✅ Прогресс-бар использования
- ✅ Дата окончания действия

## 💡 Логика отображения:

```typescript
// Занятия - ВСЕГДА показываются
{subscription.lessons_remaining}/{subscription.lesson_count}
"Занятий (осталось/всего)"

// Дни/активация - зависит от статуса
if (status === 'confirmed') {
  показать: daysRemaining + "Дней до окончания"
} else {
  показать: "-" + "Ожидает активации"
}

// Прогресс-бар - только для активных
if (status === 'confirmed') {
  показать прогресс-бар
}
```

## 🧪 Как протестировать:

1. Откройте: `http://localhost:3002/profile`

2. **Для pending абонементов** (ожидают подтверждения):
   - ✅ Видите: **"1/1 Занятий (осталось/всего)"**
   - ✅ Видите: **"- Ожидает активации"**
   - ✅ Видите желтый блок: **"Ожидает подтверждения"**

3. **После подтверждения админом** (статус → confirmed):
   - ✅ Видите: **"1/1 Занятий (осталось/всего)"**
   - ✅ Видите: **"30 Дней до окончания"**
   - ✅ Видите прогресс-бар
   - ✅ Видите дату окончания

## ✅ Результат:

Теперь **ВСЕ абонементы** показывают количество занятий в формате:

```
X/Y
Занятий (осталось/всего)
```

Где:
- **X** = lessons_remaining (осталось)
- **Y** = lesson_count (всего в абонементе)

Независимо от статуса! 🎉
